Teste de Elasticidade - Vercel

Este projeto tem como objetivo experimentar os conceitos de escalabilidade e elasticidade em aplicações serverless utilizando a Vercel.

Através de funções API que simulam cargas de trabalho, é possível observar como a plataforma cria e gerencia instâncias automaticamente conforme a demanda.
